## Cursor Key Variant:

Global actions: `<alt>+<key>`  
Controlling Cursor: Arrow Keys  
Going deeper into formula (where applicable): `<alt>`+`<up/down keys>`  
Formula Actions: `<key>` (will be applied on the formula which is active according to the cursor)  
Default Formula Action (equivalent to clicking on formula): `<space>`  
Change Tab: `<tab>`
